Additional integration options

With the iPost application program interface (API) you can seamlessly integrate your applications to output directly to iPost, creating a fully integrated mail process.

Modern businesses need flexibility and accessibility, which is why iPost allows you to upload any file type from your hard drive or cloud storage tool.

That means that you can send anything from legal correspondence to direct mailing campaigns to employee payslips without even leaving your desk.

Thanks to iPost’s API, you can also integrate it to your organisation’s CRM, accounting system or any other document-producing software.

Sending sensitive or confidential material with iPost is as safe, if not safer, than through traditional means. Documents are printed and enveloped at True cloud-based web app fulfilment and distribution centre (part of the DHL Group), which complies with all relevant security and data protection regulations. Delivered by Royal Mail.

Unlike existing mailing software systems, iPost is a cloud-based web app. No software downloads or print drivers are required, avoiding the common problem of viruses and security breaches.

What kind of files can iPost send?

iPost supports virtually every file type you could want to send in the mail from any hard drive or cloud platform.
